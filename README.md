# AI IN A BOX :Flappy Bird
Environment for your AI agent run,

### Prerequisites
* Python 3.x
* docker
* docker-compose

### Installing
* clone the repo
* run docker-compose up


## How to use

Write your code inside modules/agent.py in python and run the docker to see it in action
